# chat_memory_demo

A simple SQLite + Python project to record and view chat history.

## Features

- Create a local SQLite database
- Insert chat messages (user, question, answer)
- Retrieve full chat history

## Files

- `db.py`: Handles all database operations
- `main.py`: Demo insert and read operations

## How to Run

```bash
python main.py
```
